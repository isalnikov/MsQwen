# Веха 10: Интеграция с Qwen Code CLI

## Связь с C4 моделями
- **Level 1 (Context)**: Qwen CLI — внешняя система в контексте
- **Level 3 (Components)**: QwenService — компонент контейнера
- **Data Flow**: Поток данных от News → Qwen CLI → AnalysisResult

## Цель
Реализовать интеграцию с Qwen Code CLI для анализа новостей с передачей ВСЕХ метаданных согласно C4 моделям.

## Важные принципы
- **Все метаданные**: views, forwards, reactions, likes, engagement_score передаются в Qwen
- **ProcessBuilder**: Вызов Qwen CLI через ProcessBuilder
- **Кеширование**: Spring Cache с TTL для результатов
- **Приоритизация**: Новости с высоким engagement_score анализируются первыми
- **Batch анализ**: Несколько новостей за один вызов
- **Java 25+**, text blocks для промптов
- **Без Lombok**, чистый код
