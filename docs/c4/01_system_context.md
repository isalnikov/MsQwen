# C4 Level 1: System Context Diagram

## Обзор
MsQwen — система для парсинга новостей из публичных Telegram каналов и их анализа через Qwen Code CLI.

## Акторы

| Актор | Описание |
|-------|----------|
| **Пользователь Telegram** | Устанавливает бота, создает промпты, добавляет каналы, получает анализ новостей |
| **Администратор** | Управляет системой через CLI и REST API, очищает таблицы данных |

## Внешние системы

| Система | Описание |
|---------|----------|
| **Telegram API (t.me/s/)** | Публичные каналы — источник новостей, парсятся через Jsoup |
| **Qwen Code CLI** | AI агент для анализа новостей, вызывается через ProcessBuilder |
| **H2 Database** | Локальная база данных для хранения всех сущностей |

## Потоки данных

1. **Пользователь → MsQwen**: Команды бота (/start, /parse, /analyze, /news, /prompts, /channels, и др.)
2. **MsQwen → Telegram API**: HTTP запросы к t.me/s/ для парсинга новостей
3. **MsQwen → Qwen CLI**: Вызов CLI с промптом содержащим метаданные новостей
4. **MsQwen → H2 Database**: Сохранение пользователей, промптов, каналов, новостей, результатов анализа
5. **Администратор → MsQwen**: CLI команды и REST API запросы для управления
